# Telegram CRM & Lead Generation Bot

An asynchronous Telegram bot designed for automated lead generation and seamless integration with Google Sheets as a lightweight CRM system.

## 🚀 Features
- **Interactive Survey:** FSM-based data collection.
- **Google Sheets API:** Real-time lead storage.
- **Asynchronous Architecture:** Built with `aiogram 3.x`.

## 🛠️ Tech Stack
- Python 3.11+
- aiogram
- gspread
- python-dotenv
